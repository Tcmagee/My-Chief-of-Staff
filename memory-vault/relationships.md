## Per-Person Template

For each person tracked, maintain these fields:
- **Identity:** Name, title, team, reporting chain
- **Relationship type:** Manager / Direct report / Peer / Stakeholder / Skip-level / Cross-functional
- **Interaction frequency:** Daily / Weekly / Biweekly / Monthly / Sporadic
- **Communication style:** How they prefer to receive information
- **What they care about:** Priorities, pressures, motivations
- **Sentiment:** Warm / Neutral / Cool / At-risk (with evidence)
- **Open commitments:** What I owe them / What they owe me
- **Last meaningful contact:** Date and topic
- **Notes:** Anything else the agent should know
